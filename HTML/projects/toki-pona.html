Toki Pona

HTML, CSS, JS, Vue.js, Bootstrap

This project is a simple single page application built using Vue.js and Bootstrap. I have always been fascinated by the history of and usage of the conlang Toki Pona, and so I figured I would make a site about it. At present, the site gives an introductory explanation of what the language is, its history, and its purpose. In addition, it includes interactive flashcards for each of Toki Pona's essential vocabulary words, so that the user can learn the words themself if they so choose.

This site currently includes the use of 5 Vue components: TopBar.vue, About.vue, Flashcard.vue (this contains the logic for each individual flashcard), FlashcardView.vue (this contains the logic for switching between the flashcards, as well as how to display them), and Footer.vue (this includes links to the official Toki Pona website, as well as the Wikipedia article for the language). In the future, I would like to make a number of improvements and expansions to the site. I wish to add more pages for different topics about Toki Pona, with Vue routing between the pages. Also, I would like to improve the overall styling/animations, as well as the interactivity of the flashcards (save to local storage, study sessions, etc).
