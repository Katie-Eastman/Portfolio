Pixie Potions

HTML, CSS, JS, Bootstrap

Pixie Potions is a website for a fake business of my own making. My aim with this site was to solidify various web design principles in my skillset. I used Bootstrap to make the site mobile responsive, and I used vanilla JavaScript and CSS to implement a dark/light theme toggle for the user. Each page of the site has different content appropriate for a business site, whilst sticking to the overall look and theme. Additionally, I implemented a simple particle effect that follows the user's cursor, similar in concept but different in appearance to the one I used for my portfolio site.

In the future, I would like to clean up some of the UI for visual clarity/professionalism. I would also like to add a back-end to the contact page so that it actually works and in order to practice and demonstrate my server programming skills. I also think it would be fun to implement some 'ARG' elements to the site in order to express myself creatively and tell a more compelling story within the site itself.
