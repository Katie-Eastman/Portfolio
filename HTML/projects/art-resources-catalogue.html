Art Resources Catalogue

HTML, CSS, JS, Bootstrap, JQuery

This project is my personal ongoing collection of free art resources. It is a project that I have wanted to create for quite some time, and so learning to webdev skills necessary to bring it to life was really exciting. I created this site so that I would have a way to organize and share some of my favorite resources with other artists. Every resource in the collection is free to use and accessible on the web.

I used Bootstrap to make the site fully mobile responsive and compatible, and I also used JQuery to implement a tagging system as well as some simple animations. Each resource in the catalogue is organized in alphabetical order, and includes a link to the resource, a description of the resource, and some tags. You can use the drop-down menu to sort the resources by tag, which makes finding specific kinds of tools easier. The project itself is a work-in-progress, and I still need to fill out all the tags and descriptions. However, the code architecture is already completed and I just need to actually fill out the data for each resource manually.

In the future, besides finishing the data for each resource and getting the tagging system to work meaningfully, I would also like to add more animations, more animations/UI aspects for better visual interest and clarity, more tabs for other kinds of resources (such as pdfs of art books), and a back-end for the contact page so that it can work properly.
